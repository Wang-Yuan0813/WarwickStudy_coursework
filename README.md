Course Work
==========

A small game for WM908:Programming and Fundamental Algorithms
----------

sprites asset from https://penusbmic.itch.io/

This game has two map, a fixed and an infinite.

When player's level reaches lv.6, enter the next map.

Game save when quit from pause window.

To enhance the sense of impact, the screen will shake when the character uses skills or takes damage.
